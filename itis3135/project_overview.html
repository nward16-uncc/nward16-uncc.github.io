Client Project Overview

Project Overview

Description: HackTheBox tutorial website that guides new users.

Purpose: To teach users how they can use HackTheBox to improve their cybersecurity skills.

Intended Users: Computer Science students, cybersecurity majors, ethical hackers, or hobbyists.

Content: Information and guide for HackTheBox

Wireframe

Site Map

HOME

  1. Purpose: To introduce the user to HackTheBox
  2. Audience: Cybersecurity students, professionals, and ethical hacker hobbyists
  3. Content: Basic information and introduction to HTB
  4. Data Entry: No
  5. Links: Yes
  6. Actions: Open links in a new tab
  7. Notes: NA

GETTING STARTED

  1. Purpose: To show the user how to jump into HTB
  2. Audience: Users looking to dip their toes into HTB challenges
  3. Content: A step-by-step guide on how to start a machine
  4. Data Entry: No
  5. Links: Yes
  6. Actions: Open links in a new tab
  7. Notes: Elaborate on info from the home page

TUTORIAL

  1. Purpose: Show user how to complete their first machine on HTB
  2. Audience: User that wants to complete machine but is unsure
  3. Content: A complete writeup on an easy level machine
  4. Data Entry: Yes
  5. Links: Yes
  6. Actions: Data entry of walkthrough answers so user can try following along
  7. Notes: This will be a writeup/guide based on a HTB walkthrough

MACHINES

  1. Purpose: Introduce the user to different HTB machines
  2. Audience: User who has completed his first machine
  3. Content: Series of machine dropdown tabs with basic info
  4. Data Entry: No
  5. Links: Yes
  6. Actions: Dropdown tabs for each machine, links to machine
  7. Notes: Include multiple machines in an appealing fashion

TOOLS & RESOURCES

  1. Purpose: Show the user some common tools used to complete challenges
  2. Audience: Users that want to expand their toolset
  3. Content: Interactive gallery of some common tools
  4. Data Entry: No
  5. Links: Yes
  6. Actions: Interactive gallery, links to tools
  7. Notes: Include links to tool site / GitHub repo or tutorials

Dynamic Functionality on Website

  • Creative effects using JS
  • Collapsible sections or tabs for machine details
  • Dropdown feature for machine details
  • jQuery-UI widget and other plugins
